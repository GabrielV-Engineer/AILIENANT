## How to sign

You only need to sign **once**; it then covers all your future Submissions.

### Preferred: automated CLA check
When you open your first pull request, an automated CLA assistant will post a link. Reading and accepting it there records your signature against your GitHub identity. No PR is merged until the check is green.

### Manual fallback
If the automated check is unavailable, add a line to your pull request description (or reply to the PR) using one of the templates below.

**Individual:**
```
I have read and I agree to the AILIENANT Contributor License Agreement.
Signed-off-by: Full Name <email@example.com> (GitHub: @your-handle)
```

**Entity (contributing on behalf of an employer/organization):**
```
On behalf of <Legal Entity Name>, which I am authorized to bind, I have read
and agree to the AILIENANT Contributor License Agreement.
Signed-off-by: Full Name <email@example.com> (GitHub: @your-handle), <Title>
```
